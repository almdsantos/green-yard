🌿 Green Yard — Defenda seu jardim dos robôs invasores!

Posicione suas plantas estrategicamente nas linhas do campo e impeça que os robôs cheguem ao seu jardim. Colete gotas de água, desbloqueie novas plantas e enfrente ondas cada vez mais difíceis em 40 fases épicas!

🎮 Como jogar:
- Escolha suas plantas no almanaque
- Posicione-as no campo antes de cada fase
- Colete gotas de água para plantar mais
- Derrote todos os robôs para vencer!
